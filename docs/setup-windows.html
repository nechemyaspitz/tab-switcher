Windows Setup

Three quick steps to get Tab Switcher running on Windows. No special permissions required.

Looking for the macOS setup guide?

1

Install the browser extension

Download the extension and load it in your Chromium browser:

  1. Download and unzip the extension files
  2. Go to chrome://extensions in your browser
  3. Enable Developer mode (top right toggle)
  4. Click Load unpacked and select the unzipped folder
2

Install the Windows app

Run the installer. It sets up the native messaging bridge between the extension and the app automatically.

Download for Windows
SmartScreen warning

Windows may show a SmartScreen warning because the app is new. Click More info then Run anyway to continue.

3

Connect the extension

Open the Tab Switcher app, then link it to your browser:

  1. Go to chrome://extensions
  2. Find Tab Switcher and copy the ID (32-character string)
  3. In the Tab Switcher app, click your browser, paste the ID, and click Enable
Why is a companion app needed?

Browser extensions cannot intercept Ctrl+Tab — it's reserved by the browser. The native app captures this shortcut at the system level and communicates with the extension to show the visual switcher.

Supported browsers

Chrome, Brave, Edge, Vivaldi, Opera, and other Chromium-based browsers. The same extension works across all of them — just enable each browser in the Tab Switcher app.

Keyboard shortcuts

Ctrl + Tab

Hold Ctrl, press Tab to cycle, release to switch

Ctrl + Shift + C

Copy the current tab's URL to your clipboard

All shortcuts can be customized in the Tab Switcher app settings.
