Setup Guide

Chrome Web Store approval pending. Follow these steps for manual installation.

1

Install the browser extension

Download the extension, then load it manually in Chrome:

  1. Download and unzip the extension files
  2. Go to chrome://extensions
  3. Enable Developer mode (top right)
  4. Click Load unpacked and select the unzipped folder
2

Download the macOS app

The companion app intercepts Ctrl+Tab at the system level and displays the visual switcher.

Download for macOS
3

Grant accessibility permission

When prompted, allow Tab Switcher in System Settings → Privacy & Security → Accessibility.

4

Connect the extension

Open the Tab Switcher app, click Enable for Chrome, then enter your Extension ID:

  1. Go to chrome://extensions
  2. Find Tab Switcher and copy the ID (32-character string)
  3. Paste it into the Tab Switcher app and click Save
Why is a companion app needed?

Browser extensions cannot intercept Ctrl+Tab—it's reserved by the system. The native app captures this shortcut globally and communicates with the extension.

Keyboard shortcuts

⌃ Ctrl + Tab ⇥

Hold Ctrl, press Tab to cycle, release to switch

⌘ Cmd + ⇧ Shift + C

Copy the current tab's URL to your clipboard

All shortcuts can be customized in the Tab Switcher app settings.
