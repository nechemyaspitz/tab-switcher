Built different

01

Visual tab previews

See live thumbnails and titles as you cycle. No more guessing which tab is which.

02

Most recently used

Tabs ordered by recency, not position. Jump between your two most recent tabs instantly with a single Ctrl+Tab.

03

Native performance

A lightweight macOS companion app intercepts the shortcut at the system level. Instant response, every time.

04

Copy URL shortcut

Press Cmd+Shift+C to instantly copy the current tab's URL to your clipboard, with a sleek confirmation toast.

05

Customizable shortcuts

Remap both the tab switcher and copy URL shortcuts to whatever key combination you prefer.

06

Multi-browser support

Chrome, Brave, Edge, Arc, Vivaldi, Opera—any Chromium browser works with a single installation.

Copy any URL, instantly

One shortcut to copy the current tab's URL to your clipboard. A quick toast confirms it's done — no clicking, no address bar.

All keyboard shortcuts are fully customizable from the app settings.

Default: ⌘ + ⇧ + C
